    逻辑回归

    一些题外话

    逻辑回归，英文为（logistic regression），中的‘逻辑’与我们日常生活中所谓‘逻辑推理’中的‘逻辑’完全不同（不知道为什么中文翻译为逻辑），这里指的是logistic function，也就是sigmoid曲线。
    其实该名字源自于比利时数学家 Pierre François Verhulst ，1840年代，他在研究人口增长模型时，发现人口不会无限制呈指数增长，而是达到一定环境容量后会趋于平缓，形成一个S型曲线。为了区分于对数曲线（Logarithmic curve），他将这种曲线命名为 "courbe logistique"（法文），后来英文对应为 Logistic curve。关于 Verhulst 为什么选用 logistique 这个词，数学史界有几种说法，最主流的观点是它来源于希腊语 logistikos（意为“计算的、比例的”），因为它本质上是在计算一种对数几率（Log-odds）。总之，它创立之初就和“逻辑推理”无关。

    核心原理

    核心预测公式推导

    最早在接触机器学习是，我们往往遇到的都是线性回归模型它的目标是拟合一条连续的直线（或超平面）来预测具体的数值。模型可以表示为：

    \[z = w^T x + b\]

    然而，当我们面临分类问题，线性回归就显得力不从心了。它的输出值 \(z\) 域在 \((-\infty, +\infty)\) 之间，我们很难用一个无界的连续值来表示“概率”。为了将输出值拉回\((0, 1)\)之间，就需要sigmoid激活函数：

    \[\sigma(z) = \frac{1}{1 + e^{-z}}\]

    于是，将线性方程代入，我们就得到了逻辑回归的核心预测公式：

    \[\hat{y} = \frac{1}{1 + e^{-(w^T x + b)}} = \frac{e^{(w^T x + b)}}{1 + e^{(w^T x + b)}}\]

    参数估计：交叉熵与梯度下降

    从概率论的角度出发，我们希望模型预测正确的概率最大。假设样本独立同分布，利用极大似然估计 的思想，取对数并化简后，我们自然而然地就可以推导出对数损失（Binary Cross-Entropy）：

    \[L(y, \hat{y}) = - \left[ y \log(\hat{y}) + (1 - y) \log(1 - \hat{y}) \right]\]

    当真实标签 \(y = 1\) 时，\(L = -\log(\hat{y})\)。若预测值 \(\hat{y}\) 越接近 1，损失越接近 0；若预测值接近 0，损失趋于无穷大， \(y = 0\) 时同理。

    对于包含 \(m\) 个样本的整个训练集，成本函数 \(J\) 就是所有样本损失的平均值：

    \[J(w, b) = - \frac{1}{m} \sum_{i=1}^{m} \left[ y^{(i)} \log(\hat{y}^{(i)}) + (1 - y^{(i)}) \log(1 - \hat{y}^{(i)}) \right]\]

    梯度下降

    有了凸函数 \(J(w, b)\)，我们就可以使用梯度下降法来寻找使损失最小的权重 \(w\) 和偏置 \(b\)。

    \[dw = \frac{\partial J}{\partial w} = \frac{1}{m} X (\hat{y} - y)^T\]
    \[db = \frac{\partial J}{\partial b} = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})\]

    接下来，只需要设置一个学习率 \(\alpha\)，让参数顺着梯度的反方向不断更新即可：

    \[w := w - \alpha \cdot dw\]
    \[b := b - \alpha \cdot db\]

    算法分析

    因为逻辑回归本质上依然是一个线性分类器，它的分类边界是一条直线（或超平面）。对于线性可分的数据集，逻辑回归表现完美。但对于非线性数据 它就束手无策了 最著名的例子就是 XOR 问题。在二维平面上，正类和负类交替分布在四个象限。无论如何也无法画出一条笔直的线，将它们完美分开。一个神经元的表达能力，在这里达到了极限。

    通过逻辑回归，就可以自然地引出最简单BP神经网络的学习过程，如果将特征矩阵 \(X\) 看作输入层，它们与权重矩阵 \(w\) 进行点乘，加上偏置 \(b\)，这就构成了一个标准的、单层的、单输出的神经元。

    用代码写出其前项传播的过程，非常简洁：

    import numpy as np

A = 1 / (1 + np.exp(-(np.dot(w.T, X) + b)))

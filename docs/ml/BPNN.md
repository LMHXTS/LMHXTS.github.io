# BP神经网络

### 核心原理

在逻辑回归中，既然一个神经元只能画一条直线，那我们就把多个神经元并排放在一起，构成一个**隐藏层 (Hidden Layer)**。

假设输入特征矩阵是 $X$，我们引入一个隐藏层和一个输出层：

1. **输入层** 接收原始数据。
2. **隐藏层** 提取特征，输出 $A^{[1]}$。
3. **输出层** 根据隐藏层的特征进行最终分类，输出 $\hat{Y}$（或 $A^{[2]}$）。

但如果只做矩阵乘法 $W^{[2]}(W^{[1]}X)$，根据矩阵乘法的结合律，它等效于 $(W^{[2]}W^{[1]})X$。这说明如果不加激活函数，无论叠多少层，本质上依然是一个单层线性变换。

因此，我们需要在隐藏层引入非线性激活函数 Sigmoid
$$f(z) = \max(0, z)$$

## 二、 前向传播：优雅的矩阵之舞

在前向计算中，我们要彻底告别低效的 `for` 循环，拥抱向量化编程。为了让你在敲代码时胸有成竹，我们必须时刻盯紧**矩阵的维度**。

假设我们有 $m$ 个样本，每个样本 $n_0$ 个特征。隐藏层有 $n_1$ 个神经元，输出层有 $n_2$ 个神经元。
注意：大写字母代表包含所有 $m$ 个样本的矩阵。

**1. 隐藏层计算：**
$$Z^{[1]} = W^{[1]} X + b^{[1]}$$
$$A^{[1]} = \text{ReLU}(Z^{[1]})$$
* $X$ 的维度：$(n_0, m)$
* $W^{[1]}$ 的维度：$(n_1, n_0)$
* $Z^{[1]}$ 和 $A^{[1]}$ 的维度：$(n_1, m)$

**2. 输出层计算（以多分类 Softmax 为例）：**
$$Z^{[2]} = W^{[2]} A^{[1]} + b^{[2]}$$
$$A^{[2]} = \text{Softmax}(Z^{[2]})$$
* $W^{[2]}$ 的维度：$(n_2, n_1)$
* $A^{[2]}$（即预测结果 $\hat{Y}$）的维度：$(n_2, m)$

看，数据就像流水一样，通过矩阵乘法瞬间完成了所有样本的前向传递！

## 三、 核心灵魂：反向传播 (BP) 的矩阵推导

前向传播只是“算出一个结果”，而神经网络真正学会知识的过程，在于**反向传播**——通过计算损失函数对每个参数的偏导数（梯度），来指导参数的更新。

这是无数初学者最容易卡壳的地方，但只要我们抓住**微积分的链式法则**和**矩阵维度对齐**，一切就会迎刃而解。

假设我们使用交叉熵损失函数，对于输出层，有一个极其优雅的数学巧合：当我们把 Softmax 激活函数和多分类交叉熵损失放在一起求导时，复杂的算式会相互抵消，最终输出层的误差项 $dZ^{[2]}$ 非常简洁：

$$dZ^{[2]} = A^{[2]} - Y$$
*(其中 $Y$ 是真实标签的 One-hot 编码矩阵，维度同 $A^{[2]}$ 为 $(n_2, m)$)*

接下来，我们利用链式法则逐层回传：

**1. 输出层参数梯度：**
$$dW^{[2]} = \frac{1}{m} dZ^{[2]} A^{[1]T}$$
$$db^{[2]} = \frac{1}{m} \sum_{\text{axis=1}} dZ^{[2]}$$
*推导检验：$(n_2, m) \times (m, n_1) = (n_2, n_1)$，完美匹配 $W^{[2]}$ 的维度！*

**2. 隐藏层误差回传（核心难点）：**
误差从输出层传回隐藏层，需要经过权重 $W^{[2]}$。这里需要使用权重矩阵的转置，并乘上当前层激活函数的导数：
$$dZ^{[1]} = (W^{[2]T} dZ^{[2]}) * g'(Z^{[1]})$$
*(注意：这里 $*$ 代表矩阵元素逐个相乘（Element-wise product），$g'$ 是 ReLU 的导数)*

**3. 隐藏层参数梯度：**
$$dW^{[1]} = \frac{1}{m} dZ^{[1]} X^T$$
$$db^{[1]} = \frac{1}{m} \sum_{\text{axis=1}} dZ^{[1]}$$

到这里，BP 算法的数学引擎已经全部推导完毕。

## 四、 赢在起跑线：Kaiming 初始化的奥秘

在开始敲代码之前，还有一个决定模型成败的细节：**权重初始化**。

如果你把所有权重初始化为 0，那么隐藏层的所有神经元将会进行完全相同的计算，计算出相同的梯度，并在每次迭代后保持相同的值。这就破坏了神经网络的**非对称性 (Symmetry breaking)**。

我们通常用微小的随机数进行初始化。但是，当我们使用 ReLU 激活函数时，普通的随机初始化很容易导致信号在多层传播后急剧衰减（梯度消失）或爆炸。

**Kaiming 初始化（He Initialization）** 犹如一位精准的调音师，专门为 ReLU 量身定制。它的核心思想是：**让每一层输入和输出的方差保持一致**。通过严格的数学推导，当权重初始化满足均值为 0，方差为 $\frac{2}{n_{in}}$ 时，网络表现最好。

在代码中的实现异常简单，却能带来收敛速度的质变：

# 假设 n_in 是前一层的神经元数量，n_out 是当前层数量
W1 = np.random.randn(n_out, n_in) * np.sqrt(2.0 / n_in)
